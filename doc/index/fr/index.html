Documentation de XFce 4

Cette documentation pour XFce 4 est installée avec les différents composants qui, ensemble, constituent l'environnement de bureau. En conséquence, certains des documents listés ici peuvent éventuellement ne pas être disponibles sur votre système.

Une version de ce document est disponible en ligne à cette adresse : http://www.xfce.org/documentation/.
