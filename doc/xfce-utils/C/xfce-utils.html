Introduction

The xfce-utils package provides a taskbar, a nice about box, a run dialog and several useful scripts that are also used by other XFce 4 components such as the panel and the desktop menu.

 

Taskbar

The taskbar is either at the top or the bottom of the screen and shows all running programs. The behaviour of the taskbar can be changed from settings manager. Run xfce-setting-show taskbar, or start the settings manager from the panel. In addition to the tasklist, the taskbar also features a simple graphical pager and a notification area (also known as the system tray).

 

Run Dialog

Figure 1. Run dialog

The run dialog allows the user to launch applications. Applications can be run using a Terminal emulator.

 

About Dialog

Figure 2. About dialog

The About Dialog is used to display various informations about XFce to the user. These are basic information about the XFce 4 desktop and its components, the list of developers and contributers that made up XFce 4 and legal information.
