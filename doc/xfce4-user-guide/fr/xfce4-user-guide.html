Préface

Ce document offre un aperçu de l'environnement de bureau XFce 4 et explique comment réaliser les tâches courantes. Son rôle est d'être un point de départ pour toute personne qui souhaite utiliser Xfce 4 et de diriger vers les autres sources d'information.

La dernière version de ce document peut être trouvée à : http://www.xfce.org/documentation/xfce4-user-guide.html.
