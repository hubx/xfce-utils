Utiliser XFce4

Démarrer

Utilisez startxfce4 pour démarrer une session Xfce ; celle-ci inclut le panneau principal, le gestionnaire de fenêtres et le gestionnaire de bureau. Tous les programmes ou liens liens symboliques vers des programmes, présents dans ~/Desktop/Autostart/ seront lancés au démarrage.

Pour personnaliser le comportement de startxfce4, copiez ${sysconfdir}/xfce4/xinitrc dans votre répertoire personnel ~/.xfce4/ et éditez ce fichier.

L'emplacement du fichier xinitrc peut être différent selon votre installation. Si vous installez à partir sources, le fichier ${sysconfdir} pointe vers /usr/local/etc. Pour les paquetages compilés, il réfère souvent à /etc ; vous pouvez utiliser votre gestionnaire de paquetages pour le retrouver.

Pour terminer, le fichier xinitrc que vous avez recopié doit être exécutable : 

chmod u+x ~/.xfce4/xinitrc

 

Le bureau par défaut

L'environnement de bureau XFce 4 n'est pas un programme unique qui offre toutes les fonctionnalités, au contraire, il tente de respecter l'ancienne tradition UNIX : de petits programmes qui effectuent une seule tâche et le font bien. Un environnement par défaut sans aucun programme en cours est montré à Figure 1, « Session XFce 4 par défaut ».

Figure 1. Session XFce 4 par défaut

Au démarrage de startxfce4 plusieurs applications sont lancées :

Barre des tâches

En haut de l'écran, vous voyez la barre des tâches. Elle indique les applications ouvertes dans l'espace de travail en cours. Vous pouvez donner le focus à l'application en cliquant sur le bouton correspondant dans la barre des tâches. Cliquer à nouveau sur ce bouton cache l'application. La barre des tâches contient aussi un gestionnaire graphique représentant une vue miniature de vos espaces de travail. Vous pouvez déplacer une fenêtre d'application vers un autre espace en glissant sa représentation miniature dans le pager. L'aire de notification est invisible au départ. Elle se remplit quand une application utilise une icône de statut, par exemple, le programme de messagerie instantanée gaim.

Panel

En bas de l'écran, vous avez le panneau principal de XFce 4. Il vous permet de lancer les applications ; il possède aussi un pager graphique, une horloge et un notificateur de courrier. Certain éléments possédent un panneau de menu associé qui donne accès à plus d'applications. On ouvre les panneaux de menus en cliquant sur le bouton représentant une petite flèche proche de l'élément. En cliquant avec le bouton droit de la souris, vous pouvez modifier le contenu et les propriétés des éléments du panneau principal. Les élements du panneau comme ses poignées de déplacement possédent un menu clic-droit à partir duquel vous pouvez changer la configuration du panneau. Pour plus d'information, voyez manuel du panneau.

Gestionnaire de bureau

Le gestionnaire de bureau fournit l'image de fond et deux menus quand vous cliquez sur le fond d'écran.

Le bouton droit de la souris ouvre un menu pour lancer les applications. Lire le manuel xfdesktop pour apprendre comment modifier les choix de ce menu.

Le bouton central de la souris (ou Maj + clic gauche) ouvre la liste de toutes les applications actuellement en cours dans tous les espaces de travail. Vous pouvez activer une application en cliquant sur son nom au menu.

Gestionnaire de fenêtre

Le gestionnaire de fenêtre gère le placement des fenêtres sur l'écran et fournit les bordures et décorations des fenêtres. Il vous permet de déplacer les fenêtres en tirant sur la barre de titre et fournit les boutons de la barre de titre, pour fermer, réduire et agrandir la fenêtre, par exemple. Lire le manuel xfwm4 pour obtenir une explication complète sur le gestionnaire de fenêtres.

Gestionnaire de paramètres

Le gestionnaire de paramètres tourne en arrière-plan. Il s'assure que toutes les applications XFce 4 remettent à jour leurs paramètres quand l'utilisateur modifie une valeur dans le choix des paramètres (voir section suivante) et se charge de lire la configuration sur le disque au démarrage.

Tâches courantes

Cette section explique comment réaliser certaines tâches courantes pour démarrer plus vite votre travail avec XFce 4. Car c'est pour cela que XFce 4 est fait : vous permettre de travailler.

Lancer un programme

Panneau XFce 4

Le panneau a pour but de vous permettre un accès rapide aux applications fréquemment utilisées en les plaçant directement sur le panneau principal. Des applications moins utilisées peuvent être placées dans un menu du panneau.

Menu du bureau

Un autre moyen de lancer une application est de le faire à partir du menu souris. Lire le manuel xfdesktop pour pour apprendre comment modifier les choix de ce menu.

Dialogue exécuter

Si vous connaissez le nom du programme et si celui-ci n'est pas dans un des menus, vous pouvez utiliser la commande exécuter. Pour accèder à cette commande, tapez ou choisissez l'option 'Lancer un programme...' dans le menu du bureau.

Le dialogue garde en mémoire les dix dernières commandes exécutées avec succès.

Gérer les fenêtres et les espaces de travail

Opération courantes sur les fenêtres

Vous pouvez déplacer les fenêtres sur l'écran en tirant sur leur barre de titre. Une fenêtre peut être fermée, maximisée, enroulée ou clouée —, ce qui signifie qu'elle apparaîtra sur tous les espaces de travail — en utilisant les boutons de la barre de titre.

Un clic droit sur la barre de titre ouvre un menu vous donnant accès à toutes les opérations sur la fenêtre.

Enrouler une fenêtre signifie que seule la barre de titre restera visible. Ceci peut être réalisé avec la molette de la souris sur la barre de titre. Vers le haut, on enroule la fenêtre, vers le bas, on déroule la fenêtre.

Si vous souhaitez qu'une fenêtre ouverte au maximum ne couvre pas entièrement l'écran, vous devez ajuster les marges de l'espace de travail dans le gestionnaire de paramètres (voir ci-dessous).

Gestion des applications

Pour connaître les applications en cours, vous pouvez regarder sur la barre des tâches. Cliquer sur un bouton dans cette barre donne le focus à l'application, cliquer une nouvelle fois la cache.

Quand vous cliquez avec le bouton souris du milieu sur le fond du bureau, vous obtenez une liste des applications, classées par espace de travail. Vous pouvez activer l'application et changer d'espace de travail en cliquant sur le choix correspondant.

Le programme xfce4-iconbox peut aussi être utilisé pour garder trace des programmes en cours d'exécution.

Les espaces de travail

Vous pouvez changer d'espace de travail en cliquant sur l'un d'eux dans le pager graphique, soit dans le panneau principal, soit dans la barre des tâches. Appuyer sur ou fait défiler les espace de travail. Utiliser la molette de la souris sur le pager ou sur le fond du bureau donne le même résultat.

Pour ajouter ou supprimer un espace de travail, vous pouvez utiliser, soit le menu du bureau, soit le module de gestion correspondant (voir ci-dessous).

Utiliser le gestionnaire de paramètres

Le gestionnaire de paramètres vous donne accès aux préférences globales des applications de XFce 4. Vous pouvez le démarrer en cliquant sur son bouton dans le panneau principal, par un clic droit de la souris sur les poignées de déplacement du panneau ou par le menu souris du bureau.

De nombreux aspects de l'environnement Xfce 4 sont modifiables par ce gestionnaire. Les options accessibles sont décrites dans les manuels des composants de XFce 4. Il peut être intéressant de jeter un coup d'oeil à tous les choix pour connaître toutes les options vous permettant de créer votre meilleur environnement possible.

Aller plus loin

Ce guide de l'utilisateur vous a donné un aperçu de l'environnement XFce 4. Vous trouverez plus d'information dans les manuels de chaque composant de XFce 4.
