Utiliser Xfce4

Lancer Xfce 4

Utilisez startxfce4 pour lancer une session Xfce, qui inclut le gestionnaire de session, le panneau, la barre des tâches, le gestionnaire de fenêtres et le gestionnaire de bureaux. Tous les programmes (ou liens vers des programmes) présents dans ~/Desktop/Autostart/ seront lancés au démarrage (ce répertoire n'existe pas par défaut, il est possible que vous deviez le créer).

Par défaut, le gestionnaire de session de Xfce gère le démarrage des applications. Il vous permet de sauvegarder la session lorsque vous quittez Xfce, afin que les mêmes applications soient restaurées automatiquement lors de la connexion suivante.

Pour personnaliser le comportement de startxfce4, copiez le fichier ${sysconfdir}/xdg/xfce4/xinitrc dans votre répertoire ~/.config/xfce4/ et éditez-le. Selon la méthode d'installation utilisée, l'emplacement du fichier système xinitrc peut varier. Si vous avez installé Xfce depuis son code source, ${sysconfdir} est par défaut /usr/local/etc/. Si vous avez utilisé des binaires, c'est le plus souvent /etc/, /etc/X11, /usr/pkg/etc ou /usr/X11R6/etc; utilisez votre gestionnaire de paquets pour déterminer son emplacement.

 

Le bureau par défaut

L'environnement de bureau Xfce 4 n'est pas une seule entité qui fournit toutes les fonctionnalités, car il essaye de suivre la tradition d'UNIX qui favorise l'utilisation de plusieurs utilitaires de petite taille, qui ont chacun la charge d'une seule tâche, et l'accomplissent correctement. Cette illustration montre le bureau par défaut de Xfce 4, sans autre application. Figure 1, “La session par défaut de Xfce 4”.

Quand vous exécutez startxfce4 pour la première fois, plusieurs applications sont lancées par le gestionnaire de session de Xfce.

La barre des tâches

La barre des tâches se trouve en haut de l'écran. Elle montre les applications qui sont en cours de fonctionnement sur l'espace de travail courant. Vous pouvez focaliser la fenêtre d'un application en cliquant sur son bouton dans la barre des tâches. Cliquer une seconde fois cachera l'application. Si vous utilisez le bouton droit de la souris, un menu permettant de réaliser différentes actions sur la fenêtre de l'application apparaîtra.

La barre des tâches peut contenir un changeur de bureaux graphique qui montre une vue miniature de tous les espaces de travail, ainsi qu'une zone de notification ou "system tray".

Panneau

Le panneau se trouve en bas de l'écran. Il permet de lancer des applications, et peut contenir un changeur de bureaux graphique, une horloge, et un moniteur de messagerie. Certains boutons du panneau sont associés à un menu qui donne accès à d'autres programmes. Ces menus s'ouvrent en cliquant sur la petite flèche visible à côté des boutons.

Les modifications du panneau et des propriétés de ses éléments se font en utilisant le bouton 3 de la souris, qui fait apparaître un menu, aussi bien depuis les poignées de déplacement du panneau que depuis les différents éléments du panneau. Pour plus d'informations, consultez le manuel du panneau.

Le gestionnaire du bureau

Le gestionnaire de bureau fournit l'image de fond d'écran et les deux menu qui apparaîssent lors d'un clic sur le fond d'écran.

Le bouton 3 de la souris ouvre un menu destiné au lancement d'applications. Consultez le manuel pour apprendre comment en modifier le contenu.

Le bouton 2 de la souris (ou Shift + clic avec le bouton 1) ouvre une liste de toutes les applications en cours de fonctionnement. Vous pouvez activer une application en clliquant sur son nom dans le menu.

Le gestionnaire de fenêtres

Le gestionnaire de fenêtres est responsable du placement des fenêtres sur l'écran et fournit les décorations des fenêtres. Il permet de déplacer les fenêtres en faisant glisser la barre de titre, et fournit les boutons de la barre de titre, qui permettent par exemple de fermer, restaurer ou agrandir une fenêtre. Consultez le manuel pour des explications plus complètes sur le gestionnaire de fenêtres.

Gestionnaire de paramètres

Le gestionnaire de paramètres fonctionne en tâche de fond. Il s'assure que toutes les applications de Xfce 4 mettent leurs paramètres à jour quand l'utilisateur modifie quelque chose depuis sa fenêtre de dialogue (cf la section suivante) et se charge de lire l'ensemble des paramètres au démarrage de Xfce. Consultez les manuels du gestionnaire de paramètres et de ses plugins pour plus d'explications.

Les tâches courantes

Cette section explique comment réaliser certaines tâches courantes afin de commencer rapidement à travailler avec Xfce 4. C'est en effet pour cela que Xfce 4 a été conçu, vous permettre de travailler.

Démarrer des applications

Le panneau Xfce 4

Le panneau est conçu pour permettre un accès rapide aux applications les plus fréquemment utilisées, en les plaçant sur le panneau principal. Les applications utilisées moins souvent peuvent être placées dans un des menus du panneau.

Le menu du bureau

Le menu du bureau propose une autre méthode pour démarrer des applications. Lisez le manuel du gestionnaire de bureau pour plus d'informations sur la façon de modifier le contenu du menu.

La fenêtre de lancement

Si vous connaissez le nom d'un programme et qu'il ne se trouve pas sur le panneau ou dans le menu du bureau, vous pouvez utiliser la fenêtre de lancement. Pour l'ouvrir, tapez Alt+F2 ou choisissez l'option Exécuter... dans le menu du bureau.

Cet utilitaire garde en mémoire les 10 dernières commandes lancées avec succès.

Gérer les fenêtres et les espaces de travail

Opérations courantes sur les fenêtres

Vous pouvez déplacer les fenêtres sur l'écran en faisant glisser leur barre de titre. Une fenêtre peut être fermée, cachée, agrandie, enroulée et collée — ce qui signifie présente sur tous les espaces de travail — en utilisant les boutons de la barre de titre.

Un clic avec le bouton 3 sur la barre de titre ouvre un menu qui donne accès à toutes les actions possibles sur la fenêtre.

La roulette de la souris permet d'enrouler une fenêtre, c'est à dire la réduire à la taille de sa barre de titre, en l'actionnant alors que le pointeur de la souris est au-dessus de la barre de titre. Un mouvement vers le haut enroule la fenêtre, un mouvement vers le bas la déroule.

S vous voulez que les fenêtres agrandies ne couvrent pas l'intégralité de l'écran, vous pouvez régler les marges des espaces de travail depuis la fenêtre correspondante dans le gestionnaire de paramètres (voir ci-dessous).

Gestion des applications

Pour voir quelles applications sont en cours de fonctionnement vous pouvez regarder la barre des tâches. Cliquer sur l'un des boutons de la barre des tâches donne le focus à la fenêtre qui lui est associé. Cliquer à nouveau cache la fenêtre.

Un clic avec le bouton du milieu de la souris sur le fond d'écran fait apparaître une liste des fenêtres, classées par espace de travail. Vous pouvez alors activer une fenêtre en cliquant sur son nom.

L'application xfce4-iconbox peut aussi être utilisée pour suivre la trace des applications.

Espaces de travail

Vous pouvez changez d'espace de travail en cliquant dans le changeur de bureaux graphique, dans le panneau ou dans la barre des tâches. Presser Ctrl+Alt+LeftArrow ou Ctrl+Alt+RightArrow fait passer successivement d'un espace à l'autre. Utiliser la roulette de la souris au dessus du fond d'écran produit le même effet.

Pour ajouter ou supprimer un espace de travail, vous pouvez utiliser la liste des fenêtres accessible par un clic avec le bouton 2 sur le fond d'écran, ou encore le gestionnaire de paramètres (voir ci-dessous).

Utilisation du gestionnaire de paramètres

La fenêtre du gestionnaire de paramètres donne accès aux préférences globales des composants de Xfce 4. Vous pouvez la lancer en cliquant sur son lanceur sur le panneau, depuis le menu du bureau ou encore en exécutant xfce-setting-show.

Plusieurs plugins permettant de modifier les préférences de l'environnement sont disponibles. Consultez les différents manuels pour plus d'informations. Vous devriez jeter un coup d'oeil à toutes les fenêtres de configuration pour découvrir les options permettant la création de l'environnement de travail qui vous convient le mieux.

Pour aller plus loin

Ce guide de l'utilisateur ne donne qu'un aperçu de l'environnement de bureau Xfce 4. Plus d'informations sont disponibles dans les manuels consacrés à chaque composant.
